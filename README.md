
  This is the source code for my personal portfolio site. Built with React, Typescript, Vite, Gh-pages, Figma Make, and Google Analytics

  Note to self: To deploy changes to this site next time: Make the changes. Commit changes on main branch. Then run 'npm run deploy' locally. The internal github actions workflow will be triggered, and the changes will be reflected once the action completes. 

  ### Execution

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
